# D20 解决方案 · PM 视角草案

## 1. 产品定义与技术底座

本产品基于 **ATS3625 + CS43131** 架构，定位为高性能便携式音频解码器。

### 核心参数

| 参数 | 规格 |
| --- | --- |
| DAC 芯片 | Cirrus Logic CS43131 x2 |
| 主控 | ATS3625 |
| PCM 支持 | 最高 384kHz/32bit |
| DSD 支持 | DSD256 Native |
| 输出接口 | 3.5mm + 4.4mm 平衡 |

## 2. 差异化策略

### 2.1 硬件差异化
- 双独立晶振设计
- 高品质运放（SGM8261/8262）
- 低噪声电源（LM27762）

### 2.2 软件差异化
- 多套 EQ 预设
- DSP 音效（待确认）
